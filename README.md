## Project Overview
This repository contains my Digital Literacy Project completed as part of the CSE0001 course. The main aim of this project is to understand how to use digital tools effectively, build a basic professional online presence, communicate properly in a digital environment, and stay aware of common cyber threats.

Through this project, I worked on different tasks that helped me gain practical experience with platforms like GitHub, LinkedIn, HackerRank, and Google Forms.

---

---

## module  Summary

### Module 1 – Digital Literacy Infographic
For this task, I created an infographic using Canva to explain basic concepts of digital literacy. It includes topics like safe internet practices, useful tools for students, and maintaining a professional online presence. I focused on making the design simple and easy to understand.

---

### Module 2 – Digital Portfolio
I created profiles on GitHub, LinkedIn, and Kaggle. These platforms are useful for building a professional identity. GitHub helps in sharing projects, LinkedIn is useful for networking, and Kaggle helps in improving technical skills.

---

### Module 3 – Platforms Exploration
I completed a beginner-level coding problem on HackerRank, which helped me understand basic programming logic. I also created a Google Form quiz with five questions related to digital literacy and collected responses.

---

### Module 4 – Email Etiquette
In this task, I wrote two professional emails — one for requesting an assignment extension and another for applying for an internship. I also created a checklist of social media do’s and don’ts to promote responsible online behavior.

---

### Module 5 – Cybercrime Awareness
I studied phishing as a cybercrime case. I learned how attackers trick users into sharing personal information and what precautions can be taken. I also created a prevention checklist with safety tips.
